## 技术选型

### WebView 框架

| 框架 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| Tauri | 轻量级、性能好、安全 | 生态较小 | ⭐⭐⭐⭐⭐ |
| Electron | 生态成熟、功能丰富 | 体积大、内存占用高 | ⭐⭐⭐ |
| .NET MAUI | .NET 生态、跨平台 | 学习曲线陡 | ⭐⭐⭐ |

**推荐**：Tauri

### 前端框架

| 框架 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| React | 生态成熟、组件丰富 | 学习曲线 | ⭐⭐⭐⭐ |
| Vue | 易学、轻量级 | 生态较小 | ⭐⭐⭐⭐ |
| Svelte | 性能好、体积小 | 生态较小 | ⭐⭐⭐ |

**推荐**：React 或 Vue（根据团队熟悉度）

### 后端框架

| 框架 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| Python (FastAPI) | 简单、快速开发 | 性能一般 | ⭐⭐⭐⭐ |
| Node.js (Express) | 前后端统一、异步 | 单线程 | ⭐⭐⭐ |
| Rust (Actix) | 性能好、安全 | 学习曲线陡 | ⭐⭐⭐⭐ |

**推荐**：Python (FastAPI) 或 Rust (Actix)
